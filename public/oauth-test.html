OAuth 로그인 테스트

Google로 로그인 GitHub로 로그인

사용 방법

  1. 위 버튼을 클릭하여 OAuth 로그인을 시작합니다.
  2. 인증이 완료되면 콜백 URL로 리다이렉트됩니다.
  3. 응답으로 JWT 토큰을 받게 됩니다.

콜백 URL

  • Google: http://localhost:3000/api/auth/google/callback
  • GitHub: http://localhost:3000/api/auth/github/callback

참고: Swagger UI에서는 OAuth 플로우를 직접 테스트할 수 없습니다. 브라우저에서 위 버튼을 클릭하거나 URL을 직접 입력하여 테스트하세요.
